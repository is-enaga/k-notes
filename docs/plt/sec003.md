## 2次元カラーメッシュ pcolormesh
```python
plt.pcolormesh(<2D data / x,y,z>, ...)
 ```



### メッシュ境界線
```python
plt.pcolormesh(
    ...,
    edgecolors = '<none, etc.>'
    ...)
 ```


'none'にするとメッシュの境界線を描画しない．
